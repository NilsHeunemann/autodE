Adding QM Wrappers¶


Electronic structure (QM) calculations are handled in autodE by writing input files and parsing the output through a set of wrapper classes in (autode.wrappers). To add a new wrapper for e.g. TurboMole:

  1. Make a new file (autode/wrappers/turbomole.py) containing a TurboMole class.

  2. Subclass ElectronicStructureMethod and implement all abstract methods.

  3. Add a TurboMole class within ade.Config see here, with default keywords for each implemented type of calculation.

  4. Add the method to autode.methods as either a lcode or hcode for low- and high-level methods respectively.

  5. Optional: Include keywords in keyword wrappers e.g. wrappers/functionals.py, so keywords may have associated citations.

  6. Optional: Add implemented implicit solvent names to autode/solvent/solvents.py.

  7. Write tests, following the examples in tests/test_orca_calc.py.
